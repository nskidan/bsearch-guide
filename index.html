BSearch Technical Guide

Enterprise search for Berenson & Company — a dual-layer guide for understanding and building the system.

This guide serves two purposes: Understanding chapters (1–7) explain how BSearch works in plain English, with IB analogies and no code. Building chapters (8–13) are step-by-step specs you hand to Claude Code to implement each phase. Reference chapters (14–15) cover design decisions and terminology.

How to use this guide: Read Chapters 1–3 first for the big picture. Then read whichever chapters are relevant to what you are working on. When it is time to build, hand the Building chapter to Claude Code along with the corresponding Understanding chapter for context.
