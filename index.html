BSearch Technical Guide

Enterprise search for Berenson & Company — every architectural decision, pipeline step, and technical concept explained for the builder.

This guide is a comprehensive reference for understanding the BSearch system — an AI-powered enterprise search tool that finds documents across the firm's Microsoft 365 tenant and synthesizes answers from their content. Each chapter explains one aspect of the system, uses investment banking analogies where possible, and defines all technical jargon.

How to use this guide: Read Chapters 1–3 first for context and core concepts. Then explore the remaining chapters in any order based on what you need to understand.
